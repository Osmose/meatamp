Meatamp

Meatamp was made by @Osmose.

It uses Game_Music_Emu by Blargg, which is compiled to JavaScript via Emscripten. The Game_Music_Emu page has info on which file formats are supported.

Note that the music will stutter if the tab is unfocused because the audio samples are currently being generated on demand for SDL in the UI thread. This will hopefully go away in the future.

Meatamp has only been tested on Firefox Nightly thusfar. If you find bugs in other versions/browsers, feel free to file an issue, although I make no promises of issues ever getting fixed.

Meatamp is a music player for video game sound formats, such as NSF, SPC, and GBS files.

You can grab some NSF files to test this out from chipmusic.org.

Use the button to select a file to play.
